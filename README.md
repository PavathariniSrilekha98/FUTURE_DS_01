# FUTURE_DS_01  
## Business Sales Performance Analytics

### 📌 Objective
Analyze business sales data to identify revenue trends, top-selling products, category contribution, and regional performance.

### 🛠 Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers

### 📊 Key Insights
- North region generated the highest revenue.
- Electronics category contributes 71% of total sales.
- Sales declined from January to February.
- Chair is one of the top-performing products.

### 📌 Recommendations
- Strengthen marketing in high-performing regions.
- Improve strategy in low-performing regions.
- Focus on high-demand product categories.
- Monitor monthly performance trends closely.
